Word Wide Web

En informatica informatica, la World Wide Web (WWW)o red informática. Comúnmente conocida como la web, es un sistema de distribución de documentos de hipertexto o hipermedios interconectados y accesible vía Internet. Con un navegador web, un usuario visualiza sitios web compuesto de páginas web que pueden contener texto, imagenes, videos u otros contenidos multimedia, y navega a través de esas páginas usando hiperenlaces
